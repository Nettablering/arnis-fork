# Code of Conduct — Worldbuilders / Nettablering arnis-fork

This project follows the **Contributor Covenant v2.1**.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This Code of Conduct applies in all project spaces (GitHub issues, pull
requests, discussions, and any community channels) and when a contributor
officially represents the project in public.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behaviour may be
reported to the project maintainer at **yrjan.klokk@gmail.com**. All reports
will be reviewed and investigated promptly and fairly, following the
Contributor Covenant's standard enforcement guidelines.
